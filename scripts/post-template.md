# Post Template for Trend Blog

## Frontmatter Format
```yaml
---
title: "[Keyword]: [Catchy Title]"
date: YYYY-MM-DD
draft: true
tags: ["trending", "category"]
description: "Brief SEO-friendly description"
---
```

## Content Structure

1. **Hook** (1-2 sentences) - Why this is trending now
2. **Background** (2-3 paragraphs) - Context and history
3. **Current Situation** (2-3 paragraphs) - What's happening now
4. **Analysis/Opinion** (1-2 paragraphs) - Insights
5. **Conclusion** (1 paragraph) - Summary and outlook

## Guidelines

- Write in engaging, conversational tone
- Include relevant facts and data
- Add subheadings (##) for readability
- Target 800-1200 words
- Use images/embeds when relevant
- Cite sources naturally in text

## Example Title Formats

- "[Keyword]: What You Need to Know"
- "Why [Keyword] is Trending: A Deep Dive"
- "[Keyword] Explained: The Complete Story"
- "The [Keyword] Situation: Breaking Down the Latest"
